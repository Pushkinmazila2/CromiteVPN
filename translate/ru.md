# Stealth Browser (на базе Cromite)

Приватный Android-браузер на основе **Cromite** с встроенной локальной прокси-цепочкой через **Xray-core (VLESS + Reality)**.

Проект предназначен для того, чтобы весь трафик браузера проходил только через локальный прокси-слой без использования Android VPNService.

---


## 🚀 Ключевые особенности

*   **Native Xray Integration:** Ядро Xray интегрировано напрямую в сетевой слой Chromium через локальные сокеты. Весь трафик инкапсулируется внутри одного процесса.
*   **No VpnService Required:** Браузер не создает системный VPN-интерфейс. В статус-баре нет иконки «ключа», а система и другие приложения не «видят» использование прокси.
*   **VLESS + REALITY Stealth:** По умолчанию используется протокол REALITY. Для внешних наблюдателей (провайдера и DPI) ваш трафик выглядит как стандартная TLS-сессия к доверенному ресурсу (например, Microsoft или Google).
*   **Battery & Performance:** Отсутствие перехвата пакетов на уровне TUN-интерфейса снижает нагрузку на CPU и существенно экономит заряд батареи.
*   **Anti-Leak DNS:** Все DNS-запросы обрабатываются внутренним модулем Xray (`built-in DNS`) и уходят в зашифрованный туннель, исключая утечки через провайдера.
*   **Hidden Traffic:** Анализаторы трафика на устройстве (например, PCAPdroid) видят только одно соединение с IP-адресом вашего сервера и подмененным SNI.



## 🛠 Техническая реализация

В данной версии реализована полная инкапсуляция трафика:
1. Сетевой стек `Cronet` перенаправляет запросы во внутренний инстанс Xray.
2. Используется **Global Mode**: весь трафик браузера без исключения проходит через настроенный VLESS-аутбаунд.
3. Реализована защита от утечек через WebRTC (жесткая политика `ice_candidate_policy`).

## 📥 Настройка

На текущем этапе конфигурация осуществляется через интерфейс настроек браузера:
1. Откройте **Settings** -> **Privacy and Security**.
2. Выберите пункт **Xray Configuration**.
3. Вставьте ваш JSON-конфиг (поддерживается VLESS + Reality).
4. Перезапустите приложение для применения настроек.
